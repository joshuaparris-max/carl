# AAA-Indie Roadmap Status

## Current State

The project now has:

- A Python terminal prototype for systems iteration and automated testing.
- A single-file browser prototype in `index.html` for immediate visual playtesting.
- A detailed solo-dev production roadmap.
- A Phaser 3 engine decision document.
- Game feel, audio, visual art, content depth, UX, polish, accessibility, and release planning docs.

## Official Direction

Keep the Python prototype as the systems testbed.

Create a `phaser-client/` project and port the Tutorial Guild loop into Phaser with placeholder tiles and sprites.

The first Phaser milestone is complete when:

- Tutorial Guild renders as a tilemap.
- Carl, Donut, Mordecai, shopkeeper, enemy, chest, pickup, and hazards render as sprites.
- Existing JSON data loads.
- Movement, talk, open, inventory, craft, attack, use item, pickup, equip, buy, companion mode, save/load, and quest completion work.
- The Phaser build feels equivalent to the current browser prototype before adding new content.
